## LED Blinking using Delay Loop (8051)

### Problem Statement
Write an Embedded C program for the 8051 microcontroller to blink an LED using a software delay loop.

### Working
The LED is connected to pin P1.0 of the 8051 microcontroller.  
A delay is generated using nested loops inside a delay function.  
The LED is toggled ON and OFF repeatedly inside an infinite loop, creating a blinking effect.
